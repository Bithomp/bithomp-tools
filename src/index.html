Important
- Only use this tool on a trusted computer.
- Use a private browsing mode in your browser.
- Close the browser tab after using the tool and clear cache.
- Never share your secret key or mnemonic (recovery phrase) with anyone.

Features
- You can use this tool in two modes: online and offline.
- Works with secret key, mnemonic and Ledger Hardware Wallet.
- Multisignature signing in the Offline mode.
- Send a Payment: XRP or any IOU.
- Add or Remove a Trustline.
- Place or Cancel Buy/Sell trade orders.
- Add a Gravatar to your account.
- Assign a Domain to your account.
- Create, Execute or Cancel Escrow.
- Add a regular key.
- Add a list of signers (multisig).
- Add a message key.
- Set a flag to Require destination tag.
- Disallow Incoming XRP.
- Enable default rippling.
- Disable Master Key.

Offline mode:
- Another Computer or Phone will be required to submit the transaction.
- Offline mode is recomended.
- Download the index.html file from github. - Transfer it to the offline PC on the usb stick.
- Open the index.html file in any browser, private mode.
- You will be able to submit signed transaction blob by any device here: https://bithomp.com/submit

Online mode:
- Your Secret Key and/or Mnemonic is not transmitted over the network.
- Transactions get signed and encrypted by JS library and then transmitted through Web Socket.
- Connect to a secure web socket (WSS) server.
